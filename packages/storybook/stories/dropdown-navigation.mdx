## Definition
Dropdown navigation allows users to navigate from a menu that opens with a list of internal or external links.
<Canvas of={DropdownNavigationStories.Desktop} />

## Usage
### When to use 
- Use for links, whether internal to the application or external.
- Use to group links together when space is limited and when you can't display all the links next to each other.
- Use when you need to group related navigation elements under categories for more efficient organization and simplified navigation.
- Use when you need a hierarchical navigation structure.

### When not to use 
- Do not use for actions, use menu button instead.
- Do not use when you have only one link to display [1].
