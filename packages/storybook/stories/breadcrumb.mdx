## Definition
Breadcrumb supports wayfinding — making users aware of their current location within the hierarchical structure of a website [1].
<Canvas of={BreadcrumbStories.Default} />

## Usage
### When to use
- Use when you want to illustrate where a user currently is in the intricate hierarchy of the platform [2].
- Use when you have plenty of pages and sub-categories and nested navigation levels, or your navigation grows to three, four or even more levels [2].

### When not to use
- Do not use when you have only 1 or 2 levels deep [1]. Then, indicating what category a given page belongs to, or the tags associated with the page, is perfectly sufficient [2].
- Do not use when users explore data, filter tables, manage accounts or use search [2], since breadcrumbs should not reflect the user’s journey to the page [4].
