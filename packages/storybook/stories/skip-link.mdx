## Definition
Skip link is a link that provides a quick way for assistive technologies and keyboard users to move to the main content of a page.
<Canvas of={SkipLinkStories.Default} />

## Usage
### When to use
Use the skip link component to help keyboard-only users skip to the main content on a page [1].
