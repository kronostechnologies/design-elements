## Definition
Dropdown-list allows users to select one or multiple options displayed inside a listbox.
<Canvas of={DropdownListStories.Default} />

## Usage
### When to use
- The dropdown list should be mainly used for form filling.
- Use when there are between 5 and 15 options [3].
- Use when the options aren’t something that a user would know straight away [3].

### When not to use
- Do not use if you have fewer than five options, it is probably easier to use radio buttons, checkboxes, (or others), instead of using the extra click to get to all the list options [3].
- Do not use a dropdown list when typing may be faster. Typical situations include lists of states or countries [1]. Use combobox instead.
- Do not use if there are more than 15 options in a list, let the user search for their option with a combobox [3].
