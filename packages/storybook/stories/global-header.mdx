## Definition
The global header sits at the top of the application interface, consistently visible across all screens, serving as a hub for the application's branding and name. Additionally, it houses essential tools like navigation, search, notifications, and user settings.
<Canvas of={GlobalHeaderStories.Desktop} />

## Usage
- Use to prominently display the company's logo, name, or other branding elements to reinforce brand identity.
- Use to give users quick access to primary navigation links that help them move between different sections or features of your application.
- Use to display notifications in a prominent location allows users to quickly see updates or alerts related to their account or activity.
- Use to provide easy access to user settings, account information, or profile management features.
