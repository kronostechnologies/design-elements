## Definition
Menu Button allows users to open a menu with a list of interactive options (actions) [1].
<Canvas of={MenuButtonStories.Secondary} />

## Usage
### When to use 
- Use for actions that will change the application state (either front or back end) [2].
- Use when you need to show numerous options but have limited screen space available to display them in separate buttons [1].

### When not to use 
- Do not use when you have to bring the user somewhere else into or outside the application, use dropdown navigation instead [2].
- Do not use if you have enough space to display all the options upfront.
