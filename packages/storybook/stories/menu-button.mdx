## Definition
Menu Button allows users to open a menu with a list of interactive options (actions) [1].
<Canvas of={MenuButtonStories.Secondary} />

## Usage
### When to use 
- Use to display a list of action (buttons) [2].
- Use when you need to show numerous options but have limited screen space available to display them in separate buttons [1].

### When not to use 
- Do not use for navigation purposes, use dropdown navigation instead [2].
- Do not use if you have enough space to display all the options upfront.

### Custom Content
You can pass a custom React node to the `content` prop of a menu option to fully control its rendering. This allows you to use custom labels, icons, or any other React elements inside a menu item.

**Example:**
```jsx
<MenuButton
  options={[
    {
      content: <><MyCustomIcon /> <span>Custom Label</span></>,
      onClick: () => alert('Custom!'),
    },
    {
      label: 'Default Label',
      iconName: 'star',
      onClick: () => alert('Default!'),
    },
  ]}
/>
```
If `content` is provided, it will be rendered instead of the default label and icon.
