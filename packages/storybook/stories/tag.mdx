## Definition
Tags are used to display categories or properties [1] AND to perform small actions like selecting, and filtering [4]. 
<Canvas of={TagStories.Default} />

## Usage

### When to use 
- Use to label, organize, or categorize objects [2].
- Use to display counts [2].
- Use as a method of showing filtered data [3].
- Use to quickly filter data.
- Use to display more information, as a toggle-tip.

### When not to use 
- Do not use to display a status, use a Lozenge instead.
- Do not use to display notifications, use a badge instead.
- Do not use to display long text or long descriptions.

## Variants

### Default
Default tags are the ones used to label and categorize objects. 
<Canvas of={TagStories.Default} />

### Removable
Removable tags are often used to display selected filters or user inputs. 
<Canvas of={TagStories.Removable} />

### Clickable
Clickable tags are generally used for quick filters [6]. 
- Write these tags with nouns that describe the category to **include** in the results. Avoid negative phrases like « Exclude configured » [6].
