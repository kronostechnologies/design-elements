## Definition
The bento menu is a navigation component that represents items in a grid layout [2].
<Canvas of={BentoMenuStories.Desktop} />

## Usage
### When to use 
- Use to organize different products found in the same product family [1].
- Use to open a menu with different applications, solutions or functions within the same product [2].
- Use to quickly switch between apps, solutions, or functions within the same product family [2].

### When not to use 
- Do not use when you have limited options: using a bento menu may be unnecessary and could introduce unnecessary complexity [3].
