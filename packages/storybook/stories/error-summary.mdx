## Definition
The error summary is a notification component that lists all the errors from a form.
<Canvas of={ErrorSummaryStories.Default} />

## Usage
### When to use
- Use to help users understand what went wrong in their form submission.
- Use to help users fix the problem and recover from the error. 

### When not to use
- Do not use for other alerts than an error summary.

## Variants

### Default
- Use the default badge to display numbers within a container and/or to indicate a quantifiable status change [7].
