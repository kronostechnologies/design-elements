## Definition
Stepper input allows users to input a numeric value and incrementally adjust it using increase or decrease buttons.
<Canvas of={StepperInputStories.Default} />

## Usage
### When to use 
- To quickly set and specify values [2].
- To adjust small values when increasing or decreasing them requires only a few clicks [1-2].
- When users may not know exact values and only want to change the values that are relative to their current state [2].

### When not to use 
- If itâ€™s likely that the user will deviate from the default amount. Use the numeric input instead [4].
- If the user does not need to know what the limits are. Use the numeric input instead [4].
- If there is a wide range of numeric options. Use the slider instead [4].
- For continuous variables such as prices, distances, lengths, or human heights where specifying an exact value within a broad range is crucial, opt for numeric input or money input instead [1-2].

## Variants
### Default
By default a stepper input allows users to enter and modify a numeric value without thresholds. Use this variant when any value within the range of the stepper input is valid for your use case [3].
