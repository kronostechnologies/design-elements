## Definition
Side drawer is a component that slides over the main content displaying additional information or action.
<Canvas of={SideDrawerStories.Default} />

## Usage

### When to use
- Use to display actions that affect the content of the page.
- Use to display additional information or features.

### When not to use
- Do not use if supplemental content isn't suited for small areas or is part of a flow, consider loading a new page instead [3].
- Do not use to confirm an action, use a toast or a modal instead [3].
