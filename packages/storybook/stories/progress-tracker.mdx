## Definition
A progress tracker indicates progress through a sequence by breaking it up into multiple logical and numbered steps. They guide the user through a number of steps in order to complete a specified task [2]. 
<Canvas of={ProgressTrackerStories.Default} />

## Usage

### When to use
- Use when presenting content in chunks makes scanning easier for users and can improve theirÂ ability to comprehend [2].
- Use when the user is completing an unfamiliar or infrequent task [6].
- Use when a given task is complicated or has a specific sequence in the series of subtasks, we can decompose it into several steps to make things easier [1].
- Use if a form requires a lot of user input, it may be best to split the form into multiple steps [3].

### When not to use
- Do not use it when there is no sequence or progression, to begin with [3].
