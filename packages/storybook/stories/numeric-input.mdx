## Definition
Numeric input allows users to enter numeric values.
<Canvas of={NumericInputStories.Default} />

## Usage
### When to use
- Use for numeric values only.

### When not to use
- Do not use for currencies, use money input instead.
- Do not use for phone number, use phone input instead.
- Do not use if you want to incrementally increase or decrease the value, use stepper input instead.
