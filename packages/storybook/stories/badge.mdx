## Definition
A badge is a small notification component that appears on an element and indicates an unseen update.
<Canvas of={BadgeStories.Default} />

## Usage
### When to use
- Use the badge to notify the user of a change or update, close to the relevant item [4].
    - Use the default badge to display numbers within a container and/or to indicate a quantifiable status change [7].
    - Use the dot badge to notify the user that something is new or updated without showing a count [4].
- Use the badge when you need read-only status indicators or labels that are not interactive [9].

### When not tu use
- Notification badges should not be used for critical alerts.
- If you need a text label, use tags or lozenge instead.
