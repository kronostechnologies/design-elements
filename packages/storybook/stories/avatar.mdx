## Definition

Avatar is an atom that represents a user or an entity, that has the goal of **creating connections and humanizing online communities** [2][1]. <br />
<br />
## Usage

### When to use

- Use when you want to provide a quick and easy way to identify users, since avatars help people visually organize information without having to keep track of an individual or entity name [1][4].

### When not to use

- Do not use when an icon is more suitable, for example when denoting "things" like an address, phone number, email, etc [7].
- Do not use to replace buttons.
<br />

## Variants

### User

- Text avatars display initials that represent the user's or entity's initial.
- Text avatars always display 2 characters.
<Story of={AvatarStories.LargeSizeAvatar} />
<br />

### Image

- Image avatars are used when users can upload an image to replace the text avatar.
- The image is centred in the container.
