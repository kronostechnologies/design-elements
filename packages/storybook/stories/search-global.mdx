## Definition
The main search serves as the main search component, enabling users to input their query and trigger a popup display. Interactions with the search bar do not impact the content of the page until users either press `Enter` or click on the search button.
<Canvas of={SearchGlobalStories.Default} />

## Usage
### When to use 
- Use to help users find data more efficiently within a complex or large data set [2].
- Use to help users search in the entire app (global header) [5].
- Use to help users search for more specific data (section).  

### When not to use 
- Do not use when there is small or limited amounts of data [2].
