## Definition
Radio Cards are cards containers regrouping more information than a radio button, that allows users to select a single item from a list of related options [2]. 
<Canvas of={RadioCardStories.Horizontal} />

## Usage
### When to use
- The main difference between radio cards and radio buttons is that it is used when users need additional information on available options to make an informed decision [3].
- Use to allow users to select a single option from a group of 2 to 6 Radio Cards [2].
- Use when there is a submit button before applying the result [5].

### When not to use
- Do not use it as a static card, use a card instead [2].
- Do not use when a user can select multiple options, use checkboxes instead [2].

## Variants
### Horizontal
In most cases, use the horizontal layout to maximize the real estate available [2]. 
<Canvas of={RadioCardStories.Horizontal} />

### Vertical
Use the vertical layout when limited horizontal space, or when needing to create a vertical options list [2].
