## Usage

### When to use
- Use when users need to view content for a custom or predefined date range.
- Use when date precision is critical to the functionality, such as reports.

### When not to use
- Do not use when the range of dates spans over decades, making manual selection complex.
- Do not use to filter a calendar view. Use the Menu Button instead.
