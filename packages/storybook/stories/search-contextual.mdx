## Definition
Live search allows users to filter elements below the component directly by typing into the field. 
<Canvas of={SearchContextualStories.Default} />

## Usage
### When to use 
- Use to help users find data more efficiently within a data set [2]. 
- Use when it affects the content of the page directly when the user types characters in the field.
- Use in sections where the UI is already upfront or when the content is not too heavy.

### When not to use 
- Do not use when there is small or limited amounts of data [2]. 
- Do not use if you need to have a popup list that appears, since users donâ€™t have to make a selection to filter the options.
