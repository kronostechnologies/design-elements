## Definition
The Filter Single-Select component allows users to filter a category by choosing one mutually exclusive option.

<Canvas of={FilterSingleStories.Default} />

## Usage

### When to use
- Use to filter content from a predefined set of options.
- When users need to make a quick selection from a predefined set of options.
- Use when users can only choose between mutually exclusive options [1].
    - Ex: Gender = Female, Male, Unknown.

### When not to use
- Do not use if users need to select multiple options simultaneously (use a filter multi-select instead).

## Variants
### Default
This variant is used when the list of options is 10 or less.
<Canvas of={FilterSingleStories.Default} />

### With Search
This variant is used automatically when the list of options exceeds 10.
