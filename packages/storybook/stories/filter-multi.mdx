## Definition
The Filter Multi-Select component allows users to filter a category by choosing multiple options.

<Canvas of={FilterMultiStories.Default} />

## Usage

### When to use
- Use to filter content from a predefined set of options.
- Use when users must view or analyze results based on several selected options simultaneously.
- Use when users deal with large datasets, as it offers precise filtering options [10].
- Use when your app struggles with speed. The "Apply" button gives users control over when to trigger searches. This prevents frustration from frequent loading delays and allows users to set multiple filters before searching [10].

### When not to use
- Do not use if users need to select single options (use a filter single-select instead).


## Variants
### Default
This variant is used when the list of options is 10 or less.
