# Shadow Wrapper

The `ShadowWrapper` provides a **shadow DOM** to prevent components from getting affected / overwritten by surrounding code. This component will most likely be used in already existing apps.

## Import

```js
    import { ShadowWrapper } from '@equisoft/design-elements-react'
```

## Props

<Props of={ShadowWrapper} />

## Example

Inspect this button component to see it's **shadow DOM** parent.

<Preview>
    <Story name="normal">
        <ShadowWrapper>
            <Button label="Primary" buttonType="primary" />
        </ShadowWrapper>
    </Story>
</Preview>

You can decide the host element tag by passing it to **ShadowWrapper** via the `tagName` prop. The default host element is a div.

```jsx
<ShadowWrapper tagName="section">
    <Button label="Primary" buttonType="primary" />
</ShadowWrapper>
```
