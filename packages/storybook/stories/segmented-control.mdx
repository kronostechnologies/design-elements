## Definition
A segmented control is a horizontal set of 2 to 5 exclusive segments, each functioning as a mutually exclusive button. Segmented controls provide closely related choices that affect an object, state, or view [2][3][4]. 
<Canvas of={ToggleButtonGroupStories.Default} />

## Usage
### When to use
- Use when the change has an immediate effect on the UI [2].
- Use when the options function as mutually exclusive buttons [2].
- Use when there is a default value [1].

### When not to use
- Do not use in forms. Use checkboxes or radio buttons instead [2].
- Do not use if you want to change a system state, use a toggle switch instead.
- Do not use to alternate between views within the same context, use Tabs instead [4].
