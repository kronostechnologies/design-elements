## Definition
Segmented control is a row of 2 to 5 exclusive buttons where only one can be selected at a time. They provide closely related choices to control an object, state, or view.
<Canvas of={SegmentedControlStories.Default} />

## Usage
### When to use
- Use when the change has an immediate effect on the UI [2].
- Use when the options function as mutually exclusive buttons [2].
- Use if your options are short and consistent [4].
- Use when there is a default value [1].
- Use to restrict large groups of content or to sort related content [5].
- Use to provide closely related choices that affect an object, state, or view [6].

### When not to use
- Do not use in forms. Use checkboxes or radio buttons instead [2].
- Do not use if you want to change an option's state (on/off), use a toggle switch instead.
- Do not use to navigate content, use Tabs instead [4].
