## Definition
The password creation input allows users to create a password with instant feedback indicating if the typed password is secure enough.
<Canvas of={PasswordCreationInputStories.Default} />

## Usage
### When to use
- Use to help users create a new password.
- Use to display all the conditions necessary to create or change a password.

### When not to use
- Do not use for anything else than a password creation or modification.

## Variants
### Default
<Canvas of={PasswordCreationInputStories.Default} />

### With Manual Validation
When `liveValidation` is set to `false`, validation rules stay grey and the strength bar is hidden. Use the `validateField` callback to receive a trigger function that you can call to manually validate (e.g., on form submit). When triggered, the validation rules display in red/green states, but the strength bar remains hidden.
