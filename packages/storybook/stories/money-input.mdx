## Definition
Money input allows users to enter numeric values that represent an amount of money in a particular currency.
<Canvas of={MoneyInputStories.FrenchLocale} />

## Usage
### When to use
- Use when you have to enter or edit money values.

### When not to use
- Do not use for anything else than a money value. 

## Variants
### French Currency Display
When the component displays a CAD or USD currency in French, the currency symbol is on the right.
<Canvas of={MoneyInputStories.FrenchLocale} />

### English Currency display
When the component displays a CAD or USD currency in English, the currency symbol is on the left.
