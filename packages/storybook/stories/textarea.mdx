## Definition
Text Area allows users to input a multi-line text [1]. 
<Canvas of={TextAreaStories.Default} />

## Usage
### When to use 
- Use when users have to enter free multi-line text.

### When not to use 
- Do not use when users have to enter one line of text.
    - E.g. Name, Last Name, Organization, etc.
